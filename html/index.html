My Home Servers

Proxmox is a versatile software that allows you to run different operating systems and applications on one computer by creating virtual machines or containers. With Proxmox, I can easily host virtual machines like Windows 11 for school, an Ubuntu server for Bitwarden and Nextcloud, and manage various Docker containers for running different applications, all in one place.

By self-hosting Proxmox Backup Server, I have complete control over my backup solution. I configure incremental backups, optimize storage, and manage backups across my infrastructure. The integration with Proxmox Virtual Environment streamlines the process for my virtual machines and containers. Self-hosting Proxmox Backup Server empowers me to protect and recover my data on my terms, ensuring security and availability.

With Docker, I have the power to self-host a wide range of applications on my own server. I can easily deploy web servers, databases, content management systems, media servers, messaging tools, self-hosted cloud services, and development environments. Docker simplifies the process by packaging applications and their dependencies into containers, making it convenient to manage and run different services on my terms.

I rely on Nextcloud to host my own platform where I can store, sync, and share files, collaborate on documents, manage calendars and contacts, granting me control over my data as an alternative to services like Google Suite. Additionally, Nextcloud offers an Android app that simplifies backing up data from my phone, such as instantly uploading photos to my Nextcloud server, similar to iCloud on iPhones.

When I self-host Bitwarden, an open-source password manager, I take control of my own data. I have the power to protect my passwords with enhanced privacy and security. I can customize and integrate Bitwarden to suit my needs, eliminating the need for third-party services. By hosting it myself, I ensure that my sensitive passwords are securely stored on my own server, giving me peace of mind and the flexibility to manage them in a way that works best for me.

With Pi-hole, I have an efficient ad-blocking solution for my network. By setting my router's DNS to Pi-hole, it can block ads for my entire internet, ensuring a smoother browsing experience with fewer distractions. I can customize its settings to suit my preferences, and the user-friendly interface provides insightful statistics and logs for monitoring performance. Pi-hole helps me enjoy an ad-free browsing experience and regain control over the content I see online.

I combined Pi-hole's local DNS, Traefik Proxy, and Cloudflare for secure access to my self-hosted applications. Pi-hole assigns custom domain names, Traefik Proxy efficiently routes requests, and Cloudflare adds security and performance with CDN and SSL/TLS encryption, ensuring easy access, enhanced security, and optimized performance.

With MotionEyeOS, I can easily set up a surveillance system using IP cameras or USB webcams. Its user-friendly interface enables me to manage multiple cameras, detect motion, receive alerts, stream live video, and record footage. It's a perfect solution for my DIY home security or monitoring project, and I can easily install it on my Raspberry Pi or other single-board computers.

I use Discord and discovered the ability to create Python and Bash bots. These bots serve various functions like sending reminders, providing news and weather updates, and most importantly, monitoring the status of my servers. With my Python bot, I receive notifications if any of my servers go offline or experience connection issues. This automation allows me to stay organized, well-informed, and promptly address any server-related concerns.

Self-hosting Gitea gives me full control over my Git repositories and data, providing customization, flexibility, and cost-effectiveness. I can tailor Gitea to my needs, leverage my infrastructure for optimal performance, and upload large projects without space limitations. With self-hosting, I have the freedom to manage codebases effortlessly and maintain privacy and ownership over my repositories.
